# Tutor AI — Adaptive Learning Companion

> *The textbook that talks back. A cocapn vessel for education.*

## What It Is

A repo-native AI tutor that grows with the student. It remembers every question, every mistake, every breakthrough. After a semester, it knows the student better than any teacher.

## Features

- 📚 **Domain tracking** — separate vessels for each subject
- 🎯 **Mistake patterns** — knows what you get wrong and why
- 🧩 **Socratic method** — asks questions instead of giving answers
- 📊 **Learning style detection** — adapts to how you learn best
- 🔄 **Spaced repetition** — knows when to review and when to move on

## Deploy

```bash
git clone ./.git
cd tutor-ai
npm install
npx wrangler dev
```

## The Accumulation Advantage

After 100 sessions, Tutor AI knows:
- Which concepts you struggle with and which come easily
- What analogies work best for your learning style
- When you're guessing vs. truly understanding
- The optimal review schedule for long-term retention

No traditional tutor can compete with accumulated context. Time is the competitive advantage.
